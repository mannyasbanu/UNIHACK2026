# UNIHACK - News Sentiment to Trading Signal

A hackathon project that turns financial news sentiment into trading signals and visualises strategy performance.

## Project Structure

- `frontend/` - React dashboard for charts, sentiment, signals, and backtesting visuals
- `backend/` - FastAPI backend for news ingestion, sentiment analysis, signal generation, and backtesting

## Team Split

### Frontend
- React.js
- charts and dashboard
- metrics and signal visualisation

### Backend
- FastAPI
- financial headline ingestion
- sentiment model
- trading signal logic
- backtesting engine
